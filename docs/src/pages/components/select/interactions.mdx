#### Single Select

  <tbody>
    <tr>
      <th style={{width:"33%", textAlign: "left"}}>Initial state</th>
      <th style={{width:"33%", textAlign: "left"}}>Keyboard Interaction</th>
      <th style={{width:"33%", textAlign: "left"}}>Final state</th>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Select trigger in default state  </td>
      <td>&#60;Tab&#62;</td>
      <td>Focus shifts to the trigger</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Select trigger in focus </td>
      <td>&#60;Tab&#62;</td>
      <td>Focus shifts to the next item in the page tab sequence</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Enter&#62; / &#60;Space&#62;</td>
      <td>Popover opens. Focus shifts to search (if present), otherwise the selected/first list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Arrow Down&#62;</td>
      <td>Popover opens. Focus shifts to search (if present), otherwise the selected/first list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Arrow Up&#62;</td>
      <td>Popover opens. Focus shifts to the last list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Popover is open (focus is inside the popover)</td>
      <td>&#60;Tab&#62; / &#60;Shift Tab&#62;</td>
      <td>
        <p style={{marginBottom: 4}}><strong>List only (no footer, default):</strong></p>
        <ul style={{marginTop: 0}}>
          <li>Popover closes.</li>
          <li>Focus shifts to the next or previous focusable in the page tab sequence after the trigger (or back to the trigger if there is no next focusable).</li>
        </ul>
        <p style={{marginBottom: 4}}><strong>With Select.Footer or trapFocus enabled:</strong></p>
        <ul style={{marginTop: 0}}>
          <li>Focus shifts to the next or previous focusable inside the popover, wrapping at the end or start.</li>
          <li>Focus does not leave the popover.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Enter&#62; / &#60;Space&#62;</td>
      <td>The focused item is selected, the popover closes, and focus shifts to the trigger.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option (or search when present)</td>
      <td>&#60;Home&#62;</td>
      <td>Focus shifts to search (if present), otherwise the first list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option (or search when present)</td>
      <td>&#60;End&#62;</td>
      <td>Focus shifts to the last list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option (or search when present)</td>
      <td>&#60;Arrow Down&#62;</td>
      <td>
        <ul style={{marginTop: 0}}>
          <li>Focus shifts to the next list option (wraps to the first option after the last).</li>
          <li>With search: from the last option, focus shifts to search; from search, focus shifts to the first option.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option (or search when present)</td>
      <td>&#60;Arrow Up&#62;</td>
      <td>
        <ul style={{marginTop: 0}}>
          <li>Focus shifts to the previous list option (wraps to the last option before the first).</li>
          <li>With search: from the first option, focus shifts to search; from search, focus shifts to the last option.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Popover is open (focus is not on the trigger)</td>
      <td>&#60;Escape&#62;</td>
      <td>Popover closes and focus shifts to the trigger.</td>
    </tr>
  </tbody>
</table>
