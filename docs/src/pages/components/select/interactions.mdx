### Keyboard Interactions

<br />

#### Single Select

<table style={{width: "100%"}}>
  <tbody>
    <tr>
      <th style={{width:"33%", textAlign: "left"}}>Initial state</th>
      <th style={{width:"33%", textAlign: "left"}}>Keyboard Interaction</th>
      <th style={{width:"33%", textAlign: "left"}}>Final state</th>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Select trigger in default state  </td>
      <td>&#60;Tab&#62;</td>
      <td>The trigger will be in focus but the popover will not be opened. </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Select trigger in focus </td>
      <td>&#60;Tab&#62;</td>
      <td>Focus shifts to next item in Tab sequence on the page. <br /> The popover will be excluded from the tab sequence on the page. </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Enter&#62; / &#60;Space&#62;</td>
      <td>
       Open the popover and transfer focus to search (if present) or the option with tabindex=0 (first or first-selected).
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>down arrow</td>
      <td>
       Open the popover and transfer focus to search (if present) or the option with tabindex=0 (first or first-selected).
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;up arrow&#62;</td>
      <td>
       Open the popover, and transfer focus from the trigger to the last focusable item.
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Escape&#62;</td>
      <td>
        Nothing happens (popover is closed). When the popover is open, &#60;Escape&#62; closes it and returns focus to the trigger.
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Popover is open (focus is inside the popover)</td>
      <td>&#60;Tab&#62; / &#60;Shift&#62;&#60;Tab&#62;</td>
      <td>
       Tab order when open: search (if present) → the single list option with tabindex=0 (first or first-selected; roving tabindex) → footer. Focus cycles within the popover; Tab from the list moves to the next focusable (e.g. footer) and the popover stays open.
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Enter&#62; / &#60;Space&#62;</td>
      <td>
        Select the focused item, close the popover, and return focus to the trigger.
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Home&#62;</td>
      <td>
        Focus moves to the first focusable item (search if present, otherwise first option).
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;End&#62;</td>
      <td>
        Focus moves to the last option.
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;down arrow&#62;</td>
      <td>
        <ul>
          <li>Shift focus to the next focusable item(even if it’s in a different group).</li>
          <li>If focus is on the last focusable item, shift focus to the first focusable item.</li>
          <li>
            In case of search:
            <ul>
                <li class="ml-5">If focus is on the last item, shift focus to search. </li>
                <li class="ml-5">If focus is on search, shift focus to the first focusable item. </li>
            </ul>
          </li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;up arrow&#62;</td>
      <td>
        <ul>
          <li>Shift focus to the previous focusable item.</li>
          <li>If focus is on the first item, shift focus to the last focusable item.</li>
          <li>
            In case of search:
            <ul>
                <li class="ml-5">If focus is on the first item, shift focus to search. </li>
                <li class="ml-5">If focus is on search, shift focus to the last focusable item. </li>
            </ul>
          </li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Escape&#62;</td>
      <td>
        Close the popover. Focus is now on the trigger.
      </td>
    </tr>
  </tbody>
</table>
