      <td></td>

      </td>

      <td>
       Open the popover, and transfer focus from the trigger to either the first focusable item or the search option, depending on the context.
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;up arrow&#62;</td>
      <td>
       Open the popover, and transfer focus from the trigger to the last focusable item.
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;Escape&#62;</td>
      <td>
        Nothing happens.
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td>Popover is open (The trigger will not be in focus at this instance)</td>
      <td>&#60;Tab&#62;</td>
      <td>
       Shift focus to next item in Tab sequence on the page. <br /> The popover will be excluded from the tab sequence on the page.
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;Enter&#62;</td>
      <td>
        Select the focused item and close the popover.
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;down arrow&#62;</td>
      <td>
        <ul>
          <li>Shift focus to the next focusable item(even if itâ€™s in a different group).</li>
          <li>If focus is on the last focusable item, shift focus to the first focusable item.</li>
          <li>
            In case of search:
            <ul>
                <li class="ml-5">If focus is on the last item, shift focus to search. </li>
                <li class="ml-5">If focus is on search, shift focus to the first focusable item. </li>
            </ul>
          </li>
        </ul>
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;up arrow&#62;</td>
      <td>
        <ul>
          <li>Shift focus to the previous focusable item.</li>
          <li>If focus is on the first item, shift focus to the last focusable item.</li>
          <li>
            In case of search:
            <ul>
                <li class="ml-5">If focus is on the first item, shift focus to search. </li>
                <li class="ml-5">If focus is on search, shift focus to the last focusable item. </li>
            </ul>
          </li>
        </ul>
      </td>
    </tr>
    <tr style="vertical-align: top">
      <td></td>
      <td>&#60;Escape&#62;</td>
      <td>
        Close the popover. Focus is now on the trigger.
      </td>
    </tr>
