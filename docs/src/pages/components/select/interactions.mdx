      <td>&#60;Tab&#62; / &#60;Shift Tab&#62;</td>
      <td>Focus shifts to the next/previous focusable item inside the popover, wrapping at the end or start. Focus does not leave the popover to prevent accidental dismissals.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Enter&#62; / &#60;Space&#62;</td>
      <td>
        <p style={{marginBottom: 4}}><strong>Single select:</strong></p>
        <ul style={{marginTop: 0}}>
          <li>Focused option is selected.</li>
          <li>Popover closes. Focus returns to the trigger.</li>
        </ul>
        <p style={{marginBottom: 4}}><strong>Multi-select:</strong></p>
        <ul style={{marginTop: 0}}>
          <li>Focused option is Selected/Deselected.Popover stays open.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option</td>
      <td>&#60;Arrow Down&#62;</td>
      <td>
        <ul>
          <li>Focus shifts to the next list option.</li>
          <li>If focus is on the last option, focus wraps to the search input/first option.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Arrow Up&#62;</td>
      <td>
        <ul>
          <li>Focus shifts to the previous list option.</li>
          <li>If focus is on the first option, focus wraps to the search input/last option.</li>
        </ul>
      </td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on the search input</td>
      <td>&#60;Arrow Down&#62;</td>
      <td>Focus shifts to the first list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;Arrow Up&#62;</td>
      <td>Focus shifts to the last list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus is on a list option</td>
      <td>&#60;Home&#62;</td>
      <td>Focus shifts to the search input/first list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td></td>
      <td>&#60;End&#62;</td>
      <td>Focus shifts to the last list option.</td>
    </tr>
    <tr style={{verticalAlign: "top"}}>
      <td>Focus within Popover</td>
      <td>&#60;Escape&#62;</td>
      <td>Popover closes. Focus returns to the trigger.</td>
