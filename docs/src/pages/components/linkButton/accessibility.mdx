### Maintain Minimum Target Size for Link Buttons

It is recommended to ensure the interactive area of a link button is at least **24 × 24 px**. If the visual element is smaller, expand the clickable target area to meet the **24 × 24 px minimum target size** requirement to improve accessibility and prevent accidental activation.

### Use Descriptive Labels for Link Buttons

It is recommended to use clear and descriptive copy for link buttons to explicitly communicate the action they perform, improving usability and accessibility for all users.
