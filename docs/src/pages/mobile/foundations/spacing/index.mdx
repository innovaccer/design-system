### Scale
We have defined our scale for spacing and dimensions. We recommend using these values for spacing, height, width, margin, or padding from this scale to ensure consistency across the products. 

The scale has a difference of 4px for values under 24px after which the scale has a difference of 8px. The whole scale also consists of some outliers which are 2px, 6px, 36px, and 44px.
