As the name suggests, Status Hints can be used to highlight the status of a resource in tables, lists, headers, etc.

### Appearance

Status hint comes in 5 Appearances -

#### Info


```tsx
<StatusHint appearance="info">{'Info'}</StatusHint>
```


#### Warning


```tsx
<StatusHint appearance="warning">Warning</StatusHint>
```


#### Success


```tsx
<StatusHint appearance="success">{'Success'}</StatusHint>
```


#### Alert


```tsx
<StatusHint appearance="alert">{'Alert'}</StatusHint>
```


#### Default


```tsx
<StatusHint appearance="default">{'Default'}</StatusHint>
```

### Usage

<br/>

#### Status Hint vs Badge

Status hint is used to display the status of a resource while a badge is used as a label to tag entities.
